<range-slider>

Web component for dual handle sliders — so you can select an actual range!

History

This started as a tiny polyfill for native dual-handle sliders (<input type=range multiple>), when this was still a planned feature. In fact, it became popular enough that its CSS variables appeared in the most used CSS property names in 2020 according to Web Almanac! Unfortunately, <input type=range multiple> was dropped from the spec in 2016. After almost a decade of hiatus, multirange is now resurrected for the modern web, as a web component!

Examples

Setting different initial values:

<range-slider valuelow="30" valuehigh="60"
 oninput="this.nextElementSibling.textContent = this.value"></range-slider>
